# CH32V003-USB-DevBoard

![License](https://img.shields.io/github/license/AlexanderMandera/CH32V003-USB-DevBoard)

## Description

This is a CH32V003 USB-capable dev board (WIP).

It features:
* USB-C connector
* Optional Crystal
* User Button at PC1
* User LED at PC0
* USB (bit-bang) on PD3/PD4
